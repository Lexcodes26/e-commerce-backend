## Installation 
`npm init`

`npm install mysql`

`npm install sequelize`

`npm install dotenv`
    
## Usage

In order to run the following application you would have to follow the prompts below:

`mysql -u root -p`

`Enter PW when promted`

`source db/schema.sql`

`quit`

`npm run seed`

`npm start`
